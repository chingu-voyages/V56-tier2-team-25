Surgery Status Board
Developed as a Chingu Voyage Project - Surgery Center Status Board [Project Requirements](https://github.com/chingu-voyages/voyage-project-surgerystatus)

Surgery Status Board is a web platform for real-time surgical status tracking.

This helps hospitals keep patientsâ€™ families informed and reduces uncertainty in waiting rooms. Guest users can view real-time surgical updates, surgical team members can update patient status throughout the procedure, and admins can register new patients and initiate tracking. Designed to ease emotional stress and improve communication during critical moments.
